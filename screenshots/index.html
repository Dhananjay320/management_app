📝 Responsiveness verdict
No content cut-off at any width tested. Document width matches viewport width at 375 / 1024 / 1280 / 1440. The only elements that extend past the edge are the decorative blurred gradient orbs (.ad-orb class) which are intentionally positioned for visual effect. The sidebar collapses to a hamburger menu below ~900px.

If you're seeing cut-off on YOUR window, it's likely because: (a) browser is zoomed in (Cmd+0 to reset), (b) the window is narrower than 1024 AND you're inside a feature that needs more horizontal space (Calendar weekly view, salary breakdown table). Tell me which page + window size and I can fix.
